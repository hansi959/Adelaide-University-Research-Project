This project develops a high-resolution tennis forecasting system by improving the traditional Elo rating model using a Margin of Victory Differential Analysis (MOVDA) and a tournament-weighted framework. The system includes a structured data preprocessing pipeline that cleans historical ATP match data, categorises tournament levels, and assigns player IDs for continuous rating updates. Instead of considering only match wins and losses, the model also incorporates game and set margin differences to better capture player performance. Tournament-weighted K-factors and walk-forward validation are further applied to track player skill changes over time and improve prediction accuracy across different levels of professional tennis competition.
