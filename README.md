This project implements a high-resolution tennis forecasting system by evolving the standard Elo model into a Margin of Victory Differential Analysis (MOVDA) tournament-weighted framework. The pipeline features a modular data preprocessing architecture that cleans historical ATP data, maps tournament tiers, and encodes player IDs for recursive rating updates. By moving beyond binary outcomes to incorporate game and set differentials, the research utilises tournament-weighted K-factors and walk-forward validation to capture nuanced player skill trajectories and improve predictive accuracy across different levels of professional competition.
